Politique de confidentialité

Bolamu V5.0 — Protection de vos données de santé

person 1. Données Collectées

Bolamu collecte les catégories de données suivantes :

  • check_circleDonnées d'identité : Nom, prénom, date de naissance, numéro de téléphone
  • check_circleDonnées de santé : Antécédents médicaux, prescriptions, résultats d'analyses, consultations
  • check_circleDonnées de paiement : Numéro MoMo/Orange Money (chiffré), historique des transactions
  • check_circleDonnées techniques : Adresse IP, type d'appareil, logs de connexion

database 2. Finalités du Traitement

Vos données sont utilisées pour :

  • check_circleGérer votre compte et vos rendez-vous médicaux
  • check_circleFaciliter les consultations et prescriptions électroniques
  • check_circleAssurer le suivi médical et la continuité des soins
  • check_circleAméliorer nos services (analyse anonymisée des données)
  • check_circleRespecter les obligations légales et réglementaires

schedule 3. Conservation des Données

Les données sont conservées selon les durées suivantes :

  • check_circleDonnées de compte : Tant que le compte est actif
  • check_circleDonnées de santé : 10 ans après la dernière consultation (obligation légale)
  • check_circleDonnées de paiement : 5 ans (obligation fiscale)
  • check_circleDonnées techniques : 2 ans maximum

Après suppression du compte, les données sont anonymisées conformément au RGPD.

security 4. Sécurité des Données

Niveau de sécurité :

  • check_circleChiffrement AES-256 pour toutes les données de santé
  • check_circleCommunications sécurisées via TLS 1.3
  • check_circleServeurs hébergés sur PostgreSQL Neon (certifié SOC 2)
  • check_circleAuthentification à deux facteurs (OTP) obligatoire
  • check_circleAudits de sécurité trimestriels

verified_user 5. Droits des Utilisateurs

Conformément au RGPD, vous disposez des droits suivants :

  • check_circleDroit d'accès : Consulter toutes vos données
  • check_circleDroit de rectification : Modifier vos données inexactes
  • check_circleDroit à l'effacement : Demander la suppression de vos données
  • check_circleDroit à la portabilité : Exporter vos données
  • check_circleDroit d'opposition : Refuser le traitement marketing

Pour exercer ces droits, contactez dpo@bolamu.cg.

cookie 6. Cookies et LocalStorage

Bolamu utilise :

  • check_circleLocalStorage : Stockage local du token d'authentification et du numéro de téléphone (chiffré)
  • check_circleCookies techniques : Session management, préférences utilisateur
  • check_circlePas de cookies publicitaires : Bolamu ne vend pas vos données à des tiers

Vous pouvez supprimer le localStorage via les paramètres de votre navigateur.

share 7. Transferts et Partage

Vos données ne sont partagées qu'avec :

  • check_circleProfessionnels de santé : Médecins, pharmaciens, laboratoires (avec votre consentement)
  • check_circlePrestataires techniques : Cloudinary (documents), Neon (base de données), Africa's Talking (SMS)
  • check_circleAutorités légales : En cas d'obligation légale ou judiciaire

Engagement : Aucun partage à des fins commerciales ou publicitaires.

smart_toy 8. Agent Amina / Intelligence Artificielle

L'assistant virtuel Amina utilise :

  • check_circleDonnées anonymisées : Aucune donnée personnelle n'est stockée par l'IA
  • check_circleModèle local : L'IA fonctionne côté serveur sans export de données
  • check_circleLogs limités : Seules les interactions nécessaires sont conservées

Les conversations avec Amina ne sont pas utilisées pour entraîner des modèles tiers.

contact_mail 9. Contact DPO

Pour toute question relative à vos données personnelles :

  • check_circleDélégué à la Protection des Données (DPO) : dpo@bolamu.cg
  • check_circleAdresse postale : NBA Gestion SARLU, Brazzaville, République du Congo
  • check_circleTéléphone : +242 06 973 54 18

Le DPO répondra à toute demande dans un délai de 30 jours maximum.
